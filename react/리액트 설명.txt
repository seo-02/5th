   1) react 퍼블리싱 목적=>리액트방식의 화면구현

   2) 개발로직 중심
      전제조건) react 퍼블리싱에 먼저 능숙

   3) 리액트는 새로운 패러다임의 개발방식
       지금껏 우리가 해온 개발방식과 다르므로
       기존의 사고방식으로 리액트를 이해하려고 하지 말것

▣ 리액트 준비
1. Nodejs 설치확인 => 실행창에서 node -v
2. 편집기 => Vscode (단, Go Live로 실행하지 않음)
3. 작업폴더 react를 USB에 저장 하면 실행이 안됨(외장하드는 가능)
